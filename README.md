# CrashMe

安慕希崩溃从此"不求人"

## 🤔 这是什么

这是一份适用于游戏[Minecraft](https://www.minecraft.net/zh-hans) [Java版](https://zh.minecraft.wiki/w/Java%E7%89%88)的MC报错/崩溃分析指南,旨在提供最详细的崩溃分析及解决方案,在这里,你可以看到由众多用户经验编制而成的各色教程,希望它们可以成为你开服/整合包制作路上的一份趁手资料。嘻嘻(*∩_∩*)
